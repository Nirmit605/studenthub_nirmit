
# StudentHub

This is my repository for Week 1 of the bootcamp. I built a responsive landing page from scratch for StudentHub, which is basically a centralized command center to help students organize their daily college life, schedules, and tasks. 

What I did this week:
- Made a simple navbar with working links that smoothly scroll down to the bottom section.
- Added a big hero section with a bold heading and a "Get Started" button.
- Built a features layout at the bottom using CSS Grid to showcase what the app will do.
- Added a media query in the CSS file so the whole layout stacks into a clean single column when you open it on a mobile phone.
